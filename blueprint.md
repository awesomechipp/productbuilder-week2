# Lotto Number Generator

## Overview

This application will be a simple and visually appealing lottery number generator. It will generate 6 unique random numbers between 1 and 45.

## Implemented Features

*   **UI:** A clean and modern user interface.
*   **Generator:** A button to generate the lottery numbers.
*   **Display:** A dedicated area to display the generated numbers.

## Current Plan

1.  **HTML:** Structure the web page with a title, a button for generation, and a container for the numbers.
2.  **CSS:** Style the page for a modern and clean look, including a responsive design.
3.  **JavaScript:** Implement the logic for generating unique random numbers and displaying them dynamically on the page. We will use a Web Component to encapsulate the display of the generated numbers.
